# Pdf2PowerPoint

Pdf2PowerPoint is a tool designed to convert PDF files into PowerPoint presentations. This utility simplifies the process of transforming static PDF documents into editable PowerPoint slides, making it easier to reuse and modify content.

## Features
- Converts PDF files into PowerPoint presentations.
- Preserves the layout and formatting of the original PDF.
- Easy to use and efficient.

## Prerequisites
- Python 3.7 or higher
- Required libraries (install via `requirements.txt`)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/thomas014/Pdf2PowerPoint.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Pdf2PowerPoint
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Place the PDF file you want to convert in the project directory.
2. Run the script:
   ```bash
   python convert.py input.pdf output.pptx
   ```
   Replace `input.pdf` with the name of your PDF file and `output.pptx` with the desired name for the PowerPoint file.
